## 更新机制

本机 `test-menu.ps1` 菜单选择 `[UP]` 上传最新测速结果到服务器，客户端订阅后自动获取优选 IP。

全流程 `[FULL]` 也会自动上传。

## 客户端导入

### v2rayN（Windows）

1. 订阅分组 → 订阅分组设置
2. 添加 → 备注自定义、URL 填订阅链接 → 确定
3. 更新订阅（不通过代理）

### Clash Verge（跨平台）

1. 配置 → 新建 → 类型 Remote
2. URL 填入 `https://test.hondac.top/xkdguddkdnj.txt`
3. 点击更新按钮

### Shadowrocket（iOS）

1. 首页右上角 + → 类型 Subscribe
2. URL 填入订阅链接 → 完成
3. 返回首页更新订阅

### v2rayNG（Android）

1. 左上角菜单 → 订阅设置
2. 右上角 + → URL 填入订阅链接 → 确定
3. 返回主界面下拉更新

### sing-box（通用）

```json
{
  "outbounds": [
    {
      "type": "selector",
      "tag": "proxy",
      "outbounds": [],
      "subscription": {
        "url": "https://test.hondac.top/xkdguddkdnj.txt"
      }
    }
  ]
}
```

## 常见问题

| 问题 | 解决 |
|------|------|
| 更新失败 | 关闭代理后重试（订阅链接为国内直连） |
| 节点不通 | 运行测速更新优选 IP，重新上传 |
| 链接 404 | 检查 VPS 上传服务是否运行 |
