## 更新机制

本机 `test-menu.ps1` 菜单选择 `[UP]` 上传最新测速结果到服务器，客户端订阅后自动获取优选 IP。

全流程 `[FULL]` 也会自动上传。

## 服务端部署

### 架构

```
本机 (curl POST) → Cloudflare Tunnel → VPS upload-api → /var/www/xkdguddkdnj.txt
客户端 (GET)     → Cloudflare Tunnel → VPS nginx       → /var/www/xkdguddkdnj.txt
```

### 上传接口（Python）

文件 `/opt/upload-server.py`，监听 `127.0.0.1:10083`：

- 仅接受 `POST /upload?token=xxx`
- 二进制写入 `/var/www/xkdguddkdnj.txt`
- 404 路径统一返回 200（Cloudflare 回源兼容）

### systemd 服务

```
# /etc/systemd/system/upload-api.service
[Unit]
Description=Upload API
After=network.target

[Service]
Type=simple
ExecStart=/usr/bin/python3 /opt/upload-server.py
Restart=always

[Install]
WantedBy=multi-user.target
```

### Nginx 静态文件

```
# /etc/nginx/conf.d/sub.conf
server {
    listen 127.0.0.1:10082;
    root /var/www;
    try_files $uri =404;
}
```

### Cloudflare Tunnel 路由

```yaml
# /etc/cloudflared/config.yml
ingress:
  - hostname: test.hondac.top
    service: http://127.0.0.1:10082    # 静态文件（订阅下载）
    path: ^/xkdguddkdnj\.txt$
  - hostname: test.hondac.top
    service: http://127.0.0.1:10083    # 上传 API
    path: ^/upload
  - hostname: test.hondac.top
    service: http://127.0.0.1:10081    # 其他流量兜底
  - service: http_status:404
```

## 客户端导入

### v2rayN（Windows）

1. 订阅分组 → 订阅分组设置
2. 添加 → 备注自定义、URL 填订阅链接 → 确定
3. 更新订阅（不通过代理）

### Clash Verge（跨平台）

1. 配置 → 新建 → 类型 Remote
2. URL 填入 `https://test.hondac.top/xkdguddkdnj.txt`
3. 点击更新按钮

### Shadowrocket（iOS）

1. 首页右上角 + → 类型 Subscribe
2. URL 填入订阅链接 → 完成
3. 返回首页更新订阅

### v2rayNG（Android）

1. 左上角菜单 → 订阅设置
2. 右上角 + → URL 填入订阅链接 → 确定
3. 返回主界面下拉更新
